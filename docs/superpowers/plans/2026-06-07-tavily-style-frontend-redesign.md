# Tavily Style Frontend Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rebuild the Vue frontend as a Tavily-inspired SaaS control console while preserving all existing routes, APIs, and business behavior.

**Architecture:** Keep the current Vue/Vite app and route structure. Establish a shared shell in `WorkbenchLayout.vue`, make `SubPageLayout.vue` delegate to it, centralize design tokens and shared UI classes in `global.css`, then update page templates and visible Chinese text while keeping script logic stable.

**Tech Stack:** Vue 3, Vue Router, Vite, CSS variables, `lucide-vue-next`.

---

## File Structure

- Modify `frontend/package.json`: add `lucide-vue-next`.
- Modify `frontend/package-lock.json`: generated by `npm install`.
- Modify `frontend/assets/global.css`: define the SaaS console design system and shared classes.
- Modify `frontend/components/WorkbenchLayout.vue`: implement the application shell, navigation metadata, icons, top bar, and mobile drawer.
- Modify `frontend/components/SubPageLayout.vue`: turn it into a thin wrapper around `WorkbenchLayout`.
- Modify `frontend/views/Home.vue`: build the dashboard-style home page.
- Modify `frontend/views/DataPreparation.vue`: rebuild connection and sync console UI; keep API calls stable.
- Modify `frontend/views/UserProfile.vue`: normalize search/filter toolbar, result table, and readable Chinese text.
- Modify `frontend/views/CustomerChurn.vue`: normalize filter/action/result layout and readable Chinese text.
- Modify `frontend/views/ProductReport.vue`: normalize report controls, table panels, and readable Chinese text.
- Modify `frontend/views/ProductCompletion.vue`: normalize tabs, toolbars, observation tables, calendar, poster panels, and readable Chinese text.
- Modify `frontend/views/OngoingProduct.vue`: normalize analysis controls and wide table presentation.
- Modify `frontend/views/ChannelAnalysis.vue`: normalize filter/action/result layout and readable Chinese text.
- Modify `frontend/views/NominalBuyer.vue`: normalize search/result layout and readable Chinese text.
- Optionally modify `frontend/components/FolderCard.vue` and `frontend/components/PosterTemplate.vue` only if their visible styling conflicts with the new shared UI.

Do not modify backend files for this redesign.
